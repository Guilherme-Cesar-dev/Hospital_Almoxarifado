# Hospital_Almoxarifado
Projeto de Desenvolvimento de Sistemas II durante 1 semana.

## Caso 16 – Hospital (Almoxarifado Interno)
O almoxarifado de um hospital é responsável pelo armazenamento de materiais
essenciais como luvas, máscaras, seringas e medicamentos básicos. A falta de
controle eficiente pode comprometer o atendimento aos pacientes. O sistema
deverá garantir rastreabilidade, controle de estoque mínimo e histórico das
movimentações.

## DESAFIO
- Desenvolver um sistema web ou desktop para cadastro de produtos
- Controle de entradas e saídas de estoque
- Alerta automático quando a quantidade disponível estiver abaixo do mínimo definido.

### Sistema: hospital_almox
### Banco: hospital_db
